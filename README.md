# buzzwords
A letter and category generator for the party game buzzwords. [Try it out here!](https://calvenable.github.io/buzzwords/)

## How to play
- From an audience, select two challengers.
- Press Enter when ready, and a letter and a category will appear.
- Players take it in turns to name something in the category that starts with the letter.
- The first player to fail to name an item in 5 seconds loses!
- (Optional) The winner stays on and faces a new challenger!

Press **Enter** or **Space** to advance slides. After each round, a list of possible answers will be shown.
